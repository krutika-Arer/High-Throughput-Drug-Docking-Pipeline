# High-Throughput-Drug-Docking-Pipeline
A reproducible pipeline for high-throughput molecular docking of FDA-approved drugs using AutoDock Vina, including ligand minimization, preparation, docking, and score extraction.

## 📂 Project Structure

scripts/ → pipeline scripts  
config/ → docking configuration  
data/ → user-provided input files  

---

## ⚠️ Data Note

DrugBank datasets are not included due to size and licensing restrictions.  
Users must download and place them manually.
